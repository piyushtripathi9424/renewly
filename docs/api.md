
## Provider Catalog API
- `GET /providers`: Search and filter providers (paginated)
- `GET /providers/:id`: Get provider details by ID
- `GET /providers/slug/:slug`: Get provider details by slug
- `POST /providers`: Create a new provider (Admin only)
- `PATCH /providers/:id`: Update an existing provider (Admin only)
- `DELETE /providers/:id`: Delete a provider (Admin only)
